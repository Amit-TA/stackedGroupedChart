Stacked and Grouped Bar Chart
===================

Stacked Grouped Bar Chart built using D3.js

Usage
=====
var innerColumns = {
  "column1" : ["column_name", "column_name"],
  "column2" : ["column_name"],
  "column3" : ["column_name"]
}
You can modify each group's stacks using innerColumns Object.
